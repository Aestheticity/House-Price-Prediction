# House Price Prediction

Machine Learning project that predicts house prices using the California Housing Dataset.

## Technologies
- Python
- Scikit-Learn
- Pandas
- NumPy

## Models
- Linear Regression
- Random Forest Regression

## Results
Random Forest achieved higher R² score than Linear Regression.

## Files
- House_Price_Prediction.ipynb
- house_price_model.pkl

## To generate the trained model:

1. Run the notebook
2. The notebook will create house_price_model.pkl automatically
